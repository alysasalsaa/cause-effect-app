# cause-effect-app

A master-detail UI pattern app — click a name to view their full details.

## Features

- [x] List of people displayed in a summary pane
- [x] Click a name to show full details in the adjacent pane
- [x] Click another name to refresh the detail pane
- [x] Hover highlight effect on names
- [x] Selected highlight effect (different from hover)
- [x] Selection highlight removed when a new name is clicked

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
